# Подключение проекта к GitHub

Git и GitHub CLI уже установлены. Репозиторий инициализирован, первый коммит сделан.

## Один раз: войти в GitHub

В терминале (PowerShell или Cursor):

```powershell
gh auth login --web --hostname github.com
```

Скопируйте одноразовый код из вывода, откройте ссылку в браузере, введите код и подтвердите вход.

## Создать репозиторий и отправить код

Из папки проекта выполните:

```powershell
cd e:\composeinternals
.\push_to_github.ps1
```

Скрипт создаст на GitHub публичный репозиторий `composeinternals`, привяжет его как `origin` и выполнит `git push`. Имя репозитория можно передать аргументом: `.\push_to_github.ps1 my-repo-name`.

## Чтобы LLM могла пушить автоматически

После входа через `gh auth login` GitHub CLI сохраняет учётные данные — push из Cursor (вручную или через ассистента) будет работать без дополнительного ввода пароля.
