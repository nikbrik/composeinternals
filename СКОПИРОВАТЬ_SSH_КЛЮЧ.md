# Как вручную скопировать SSH-ключ в GitHub

## Способ 1: Через проводник и блокнот

1. Нажмите **Win + R**, введите:
   ```
   %USERPROFILE%\.ssh
   ```
   Нажмите Enter. Откроется папка с ключами.

2. Откройте файл **`id_ed25519.pub`** двойным щелчком (или правый клик → Открыть с помощью → Блокнот).

3. В блокноте будет одна строка вида:
   ```
   ssh-ed25519 AAAAC3NzaC1lZDI1NTE5AAAAIJP46kbOHRiNqa2NgYLCO8NIlNREsgEPxvJWmuM5MI2h nikbrik@github
   ```
   Выделите её целиком (**Ctrl + A**), скопируйте (**Ctrl + C**).

4. Откройте в браузере: **https://github.com/settings/ssh/new**

5. В поле **Title** напишите что угодно (например: `Мой ПК`).

6. В поле **Key** вставьте скопированную строку (**Ctrl + V**). Не должно быть лишних пробелов в начале или в конце, не должно быть переноса строки — только одна строка.

7. Нажмите зелёную кнопку **Add SSH key**.

---

## Способ 2: Через PowerShell (скопировать в буфер обмена)

1. Откройте PowerShell (Win + X → Windows PowerShell, или терминал в Cursor).

2. Выполните одну команду:
   ```powershell
   Get-Content $env:USERPROFILE\.ssh\id_ed25519.pub | Set-Clipboard
   ```
   После этого содержимое ключа уже в буфере обмена.

3. Откройте **https://github.com/settings/ssh/new**, вставьте в поле **Key** (**Ctrl + V**), заполните Title и нажмите **Add SSH key**.

---

## Важно

- Копировать нужно именно файл **`.pub`** (публичный ключ), не `id_ed25519` без расширения (это приватный ключ, его никому показывать нельзя).
- В поле Key на GitHub должна быть **одна строка**, без переноса и без лишних пробелов в начале/конце.
