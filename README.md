# Compose Internals

Книга о внутреннем устройстве Jetpack Compose (оригинал на английском + русский перевод глав в файлах `chap00_ru.md` … `chap07_ru.md`).

## Подключение к GitHub (один раз)

1. В терминале: **`gh auth login --web --hostname github.com`** — войти в аккаунт GitHub (откроется браузер).
2. Затем: **`.\push_to_github.ps1`** — создаст репозиторий на GitHub и отправит код.

Подробности: [CONNECT_GITHUB.md](CONNECT_GITHUB.md).
